# WhatsApp Auto-Transcriber

A lightning-fast, privacy-focused CLI tool that automatically transcribes incoming WhatsApp voice notes on macOS and Windows using OpenAI Whisper.

## Why use this tool?

WhatsApp's native transcription is often unavailable or slow. This tool provides a superior, **100% local** alternative that works instantly the moment a voice note is downloaded, leveraging your hardware (GPU/NPU/CPU) for maximum speed and privacy.

<Note>
  **Desktop only:** This tool requires the official WhatsApp Desktop App (Store Version). It monitors the internal file system for new `.opus` audio files.
</Note>

## What you'll need

Before installing WhatsApp Auto-Transcriber, make sure you have:

- **Python 3.10 or higher** - Required for the application runtime
- **FFmpeg** - Audio processing dependency for OpenAI Whisper
- **WhatsApp Desktop** - Official Store version (not the web client)
- **3-10 GB disk space** - Depends on which Whisper model you choose

See the [Prerequisites](/prerequisites) page for detailed installation instructions.

## How it works

The transcription workflow is simple and automatic:

<Steps>
  <Step title="Start the transcriber">
    Run `wa-transcriber` in your terminal. The tool loads the Whisper model and begins monitoring your WhatsApp Media folder.
  </Step>
  
  <Step title="Receive a voice note">
    When someone sends you a voice note in WhatsApp Desktop, the file is automatically downloaded to the monitored folder.
  </Step>
  
  <Step title="Instant transcription">
    The tool detects the new file, transcribes it using your selected Whisper model, and copies the text to your clipboard.
  </Step>
  
  <Step title="Use the transcript">
    Simply paste (Cmd+V or Ctrl+V) anywhere you need the text. The transcript is also saved to daily logs for future reference.
  </Step>
</Steps>
