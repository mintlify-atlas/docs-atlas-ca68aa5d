## Overview

The `config.py` module manages application settings, persistent configuration storage, and OS-specific path detection for WhatsApp Media folders.

**Location:** `app/config.py`

    "ENABLE_MPS_FP16": False,
    "SYSTEM_MEMORY_LIMIT_FACTOR": 0.5,
    "NVIDIA_VRAM_LIMIT_FACTOR": 0.7,
    "FILE_READY_TIMEOUT": 10,
    "MANUAL_PATH_OVERRIDE": None,

### Configuration Fields

<ParamField path="MODEL_SIZE" type="str" default="turbo">
  Whisper model to use: `tiny`, `base`, `small`, `medium`, `turbo`, or `large`.
</ParamField>

<ParamField path="TRANSCRIPTION_LANGUAGE" type="str | None" default="None">
  ISO language code (e.g., `"en"`, `"es"`). `None` enables auto-detection.
</ParamField>

<ParamField path="SCAN_LOOKBACK_ENABLED" type="bool" default="True">
  Enable startup scan for missed audio files.
</ParamField>

<ParamField path="SCAN_LOOKBACK_HOURS" type="int" default="1">
  How many hours back to scan for missed files.
</ParamField>

<ParamField path="MODEL_CLEANUP_ENABLED" type="bool" default="True">
  Automatically delete unused Whisper model files.
</ParamField>

<ParamField path="MODEL_RETENTION_DAYS" type="int" default="3">
  Days to keep unused models before deletion.
</ParamField>

<ParamField path="ENABLE_MPS_FP16" type="bool" default="False">
  Enable FP16 inference on Apple Silicon (experimental).
</ParamField>

<ParamField path="SYSTEM_MEMORY_LIMIT_FACTOR" type="float" default="0.5">
  Max fraction of system RAM to use (50%).
</ParamField>

<ParamField path="NVIDIA_VRAM_LIMIT_FACTOR" type="float" default="0.7">
  Max fraction of NVIDIA VRAM to use (70%).
</ParamField>

<ParamField path="FILE_READY_TIMEOUT" type="int" default="10">
  Seconds to wait for file write completion.
</ParamField>

<ParamField path="MANUAL_PATH_OVERRIDE" type="str | None" default="None">
  Custom WhatsApp Media path. Overrides auto-detection.
</ParamField>

## Functions

### load_configuration()

Loads user settings from `config.json` and updates module-level variables.

```python
def load_configuration() -> bool
```

<ResponseField name="return" type="bool">
  Returns `True` if config loaded successfully, `False` if file doesn't exist (first run).
</ResponseField>

**Example:**

```python
from app import config

if config.load_configuration():
    print(f"Loaded config: {config.MODEL_SIZE}")
else:
    print("First run - using defaults")
```

**Behavior:**

- Returns `False` if `config.json` doesn't exist (first run)
- Reads JSON file and updates global variables
- Falls back to `DEFAULT_CONFIG` values for missing keys
- Prints warning if JSON parsing fails

**Implementation:**

```python
if not CONFIG_FILE_PATH.exists():
    return False

try:
    with open(CONFIG_FILE_PATH, "r") as f:
        user_config = json.load(f)
    
    MODEL_SIZE = user_config.get("MODEL_SIZE", DEFAULT_CONFIG["MODEL_SIZE"])
    TRANSCRIPTION_LANGUAGE = user_config.get(
        "TRANSCRIPTION_LANGUAGE", DEFAULT_CONFIG["TRANSCRIPTION_LANGUAGE"]
    )
    # ... (loads all config fields)
    return True
except Exception as e:
    print(f"⚠ Error loading config: {e}")
    return False
```

### find_default_whatsapp_path()

Attempts to auto-detect WhatsApp Media folder based on OS conventions.

```python
def find_default_whatsapp_path() -> Optional[str]
```

<ResponseField name="return" type="str | None">
  Returns path string if found, otherwise `None`.
</ResponseField>

**Detection Logic:**

#### macOS (Darwin)

```python
if SYSTEM == "Darwin":
    _mac_store = HOME_DIR / "Library/Group Containers/group.net.whatsapp.WhatsApp.shared/Message/Media"
    _mac_direct = HOME_DIR / "Library/Application Support/WhatsApp/Media"
    
    if _mac_store.exists():
        return str(_mac_store)  # Mac App Store version
    elif _mac_direct.exists():
        return str(_mac_direct)  # Direct download version
```

#### Windows

```python
elif SYSTEM == "Windows":
    local_app_data = os.getenv("LOCALAPPDATA", "")
    
    _win_store = os.path.join(
        local_app_data,
        "Packages",
        "5319275A.WhatsAppDesktop_cv1g1gvanyjgm",
        "LocalState",
        "shared",
        "transfers",
    )
    _win_legacy = os.path.join(local_app_data, "WhatsApp", "Media")
    
    if os.path.exists(_win_store):
        return _win_store  # Microsoft Store version
    elif os.path.exists(_win_legacy):
        return _win_legacy  # Legacy version
```

**Example:**

```python
path = config.find_default_whatsapp_path()
if path:
    print(f"Found WhatsApp at: {path}")
else:
    print("Could not auto-detect WhatsApp path")
```

### detect_whatsapp_path()

Sets the `WHATSAPP_INTERNAL_PATH` global variable. Prioritizes manual override, then auto-detection.

```python
def detect_whatsapp_path() -> None
```

**Priority Order:**

1. **Manual Override** - If `MANUAL_PATH_OVERRIDE` is set and exists
2. **Auto-Detection** - Calls `find_default_whatsapp_path()`

**Implementation:**

```python
global WHATSAPP_INTERNAL_PATH

# 1. Check Manual Override
if MANUAL_PATH_OVERRIDE and os.path.exists(MANUAL_PATH_OVERRIDE):
    WHATSAPP_INTERNAL_PATH = MANUAL_PATH_OVERRIDE
    return

# 2. Check Auto-Detect
WHATSAPP_INTERNAL_PATH = find_default_whatsapp_path()
```

**Example:**

```python
from app import config

config.detect_whatsapp_path()

if config.WHATSAPP_INTERNAL_PATH:
    print(f"Using path: {config.WHATSAPP_INTERNAL_PATH}")
else:
    print("Path not found - run 'wa-transcriber setup'")
```
