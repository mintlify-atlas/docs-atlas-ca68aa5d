The WhatsApp Auto-Transcriber provides a simple CLI for managing and running the transcription service.

## Command Structure

All commands follow this pattern:

```bash
wa-transcriber [COMMAND] [OPTIONS]
```

## Quick Start

<Steps>
  <Step title="First Time Setup">
    Run the interactive setup wizard to configure the application:
    ```bash
    wa-transcriber setup
    ```
  </Step>
  
  <Step title="Verify Installation">
    Check that all dependencies are installed:
    ```bash
    wa-transcriber health
    ```
  </Step>
  
  <Step title="Start Transcribing">
    Run the transcriber:
    ```bash
    wa-transcriber
    ```
  </Step>
</Steps>

## Getting Help

Display help information for any command:

```bash
wa-transcriber --help
```

<Note>
  The transcriber runs in the foreground and monitors your WhatsApp media folder continuously. Press `Ctrl+C` to stop.
</Note>
