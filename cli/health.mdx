The `health` command performs comprehensive system diagnostics to verify that all dependencies are installed and the application is properly configured.

## Usage

```bash
wa-transcriber health
```

## Memory Requirements

The health check uses these minimum memory requirements per model:

| Model | Memory Required |
|-------|----------------|
| `large` | ~10 GB |
| `turbo` | ~6 GB |
| `medium` | ~5 GB |
| `small` | ~2 GB |
| `base` | ~1 GB |
| `tiny` | ~1 GB |

<Note>
  Memory limits are calculated based on your configuration:
  - **NVIDIA GPUs**: `NVIDIA_VRAM_LIMIT_FACTOR` (default: 70%)
  - **System RAM**: `SYSTEM_MEMORY_LIMIT_FACTOR` (default: 50%)
</Note>

## When to Run Health Check

Run the health check:

- After installing the application
- When experiencing performance issues
- After changing hardware
- Before upgrading to a larger model
- When troubleshooting errors
